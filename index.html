Sentiment Analysis Grammar
based on NLTK

Course: NATRUAL LANGUAGE ANALYSIS

Individual Project

Developped a general feature based CFG (Context-Free Grammar) for English using NLTK, the feature grammar was based on Earleys Chart parser. Then built a basic sentiment analysis module based on this grammar and other open source components and enhanced it with ideas from the course.

Technologies: Python, AFINN

Substitution Encryption and Hacking Programs

Course: FOUNDATIONS OF CRYPTOGRAPHY

Group Project group of 4, role: programmer(1/1)

Developed two programs with user friendly interface, one for substitution encryption and one for breaking it using two-gram frequency of letters. The encryption program encrypts the plain-text with any non-dublicate key, and the other program automatically break the ciphertext produced by the encrption program.

Technologies: Mac OS, Eclipse, Java, Java Swing

Resolution Program to Logic Dots

Course: INTELLIGENT SYSTEMS

Group Project group of 2, role: programmer(1/2)

Developed a resolution program to the game logic dots which is a small game trying to find the right positions of the dots based on the number provided in the column and row. Search the name on the internet and you can see what it is. What we did in the project was writing a program to find the answer of the game. We used two ways to solve the problem respectively which are depth first algorithm and best first algorithm.

Technologies: prolog
